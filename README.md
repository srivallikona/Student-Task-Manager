# Student-Task-Manager
A responsive student task manager web app that allows users to add, complete, and delete tasks with data stored using browser LocalStorage
# 📝 Student Task Manager

A simple web app to manage daily tasks.

## Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Data stored using LocalStorage

## Tech Used
- HTML
- CSS
- JavaScript

## How to Run
Open index.html in your browser.
